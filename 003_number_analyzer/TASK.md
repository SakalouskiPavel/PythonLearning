# Задача 003 — «Анализатор числа»

## Цель

Научиться изменять поведение программы в зависимости от данных: использовать
условия, сравнения, логические выражения и остаток от деления.

## Условие

Создай в этой папке файл `main.py`. Программа должна запросить одно целое число и
сообщить:

1. Положительное оно, отрицательное или равно нулю.
2. Чётное оно или нечётное.
3. Делится ли оно одновременно на `3` и на `5` без остатка.

Пример для числа `15`:

```text
Введи целое число: 15
Число положительное.
Число нечётное.
Число делится и на 3, и на 5.
```

Пример для числа `-8`:

```text
Введи целое число: -8
Число отрицательное.
Число чётное.
Число не делится одновременно на 3 и на 5.
```

Формулировки можно немного изменить, если смысл сообщений сохраняется.

## Темы и ключевые слова для изучения

- условные конструкции `if`, `elif`, `else`;
- отступы и блоки кода в Python;
- операторы сравнения `>`, `<`, `==`;
- оператор остатка от деления `%`;
- логический тип `bool`, значения `True` и `False`;
- логический оператор `and`;
- условие чётности: остаток от деления на `2`;
- преобразование ввода с помощью `int()`.

## Ограничения

- Используй один вызов `input()`.
- Преобразуй введённое значение в `int`.
- Для определения знака используй одну цепочку `if` / `elif` / `else`.
- Для определения чётности используй `%`, а не перечень известных чисел.
- Проверка делимости на `3` и `5` должна содержать оператор `and`.
- Не используй циклы, функции и сторонние библиотеки — они появятся в следующих
  задачах.
- Пока можно считать, что пользователь всегда вводит корректное целое число.

## Важная деталь

Ноль является чётным и делится без остатка на любое ненулевое целое число.
Поэтому для `0` программа должна сообщить, что число равно нулю, является чётным
и делится одновременно на `3` и на `5`.

## Критерии готовности

- Файл называется `main.py` и запускается без ошибок.
- Для положительных, отрицательных чисел и нуля правильно определяется знак.
- Для любых целых чисел правильно определяется чётность.
- Делимость одновременно на `3` и `5` определяется правильно.
- Для каждой из трёх характеристик выводится ровно одно подходящее сообщение.
- Код имеет правильные отступы и понятные имена переменных.
- Ты можешь объяснить разницу между `=` и `==`, а также результат выражения
  `number % 2`.

## Самопроверка

Проверь программу как минимум на этих значениях:

| Число | Знак | Чётность | Делится на 3 и 5 |
|---:|---|---|---|
| `15` | положительное | нечётное | да |
| `-8` | отрицательное | чётное | нет |
| `0` | ноль | чётное | да |
| `30` | положительное | чётное | да |
| `7` | положительное | нечётное | нет |

## Дополнительное задание (необязательно)

Сообщи, находится ли число в диапазоне от `-100` до `100` включительно. Для
этого самостоятельно изучи цепочку сравнений вида `a <= value <= b`.
