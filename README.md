# Практическое изучение Python

Этот репозиторий — персональный практический курс Python. Обучение начинается с
небольших консольных программ и постепенно дойдёт до разработки полноценного
веб-приложения.

## Как устроено обучение

Каждая задача находится в отдельной пронумерованной папке. В ней есть файл
`TASK.md` с условиями, темами для изучения, критериями готовности и дополнительным
заданием. Решение нужно помещать в ту же папку, не изменяя `TASK.md`.

Обычный рабочий цикл:

1. Прочитать условие и самостоятельно изучить перечисленные темы.
2. Написать и запустить решение в папке задачи.
3. Сообщить мне, что задача готова к проверке.
4. Я проверю код и его запуск, отмечу сильные стороны и проблемы, дам советы и
   итоговую оценку.
5. При необходимости исправить решение и отправить его на повторную проверку.
6. После успешного завершения я создам папку со следующей задачей.

Если возникнет затруднение, можно попросить подсказку. Я постараюсь дать её
поэтапно, не раскрывая сразу готовое решение.

## Как оцениваются решения

При проверке учитываются:

- соответствие условиям и корректность результата;
- понятность и аккуратность кода;
- использование тем текущей задачи;
- обработка предусмотренных условием ситуаций;
- способность объяснить, как работает решение.

Оценка выставляется по десятибалльной шкале и сопровождается объяснением.
Идеальное решение не требуется: цель курса — устойчивый прогресс и понимание.
