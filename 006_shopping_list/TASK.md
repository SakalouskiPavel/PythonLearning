# Задача 006 — «Список покупок»

## Цель

Создать интерактивный менеджер списка покупок и научиться хранить несколько
значений в списке, добавлять и удалять элементы, искать их и перебирать в цикле.

## Условие

Создай файл `main.py`. Программа должна хранить покупки в списке и многократно
предлагать пользователю одну из четырёх команд:

```text
add     — добавить покупку
show    — показать список
remove  — удалить покупку
exit    — завершить программу
```

Названия команд вводятся на английском ровно в таком виде. После выполнения
команды программа снова запрашивает команду, кроме случая `exit`.

## Поведение команд

### `add`

Запросить название покупки. Если такого значения ещё нет в списке, добавить его
с помощью `append()` и сообщить об успехе. Если значение уже есть, вывести
сообщение о дубликате и ничего не добавлять.

### `show`

Если список пуст, вывести `Список покупок пуст.`. Иначе вывести все элементы с
номерами, начиная с единицы, а затем их общее количество.

Пример:

```text
1. Хлеб
2. Молоко
Всего покупок: 2
```

### `remove`

Запросить название покупки. Если оно есть в списке, удалить его с помощью
`remove()` и сообщить об успехе. Если его нет, вывести понятное сообщение об
ошибке. Программа не должна аварийно завершаться.

### `exit`

Вывести сообщение о завершении и выйти из основного цикла с помощью `break`.

Для любой неизвестной команды вывести `Неизвестная команда.` и продолжить
работу.

## Пример диалога

```text
Команда: add
Название покупки: Хлеб
Покупка добавлена.
Команда: add
Название покупки: Молоко
Покупка добавлена.
Команда: show
1. Хлеб
2. Молоко
Всего покупок: 2
Команда: remove
Название покупки: Хлеб
Покупка удалена.
Команда: exit
До встречи!
```

## Темы и ключевые слова для изучения

- список: `list` и литерал `[]`;
- изменяемые коллекции;
- методы `append()` и `remove()`;
- оператор принадлежности `in` и проверка `not in`;
- длина списка: `len()`;
- перебор списка циклом `for`;
- нумерация через `enumerate(items, start=1)`;
- проверка пустого списка: `if not items`;
- строковые методы `strip()` и `lower()`;
- меню на основе `while True`;
- повторение ранее изученных `break`, условий и f-строк.

## Ограничения

- Создай один пустой список до начала основного цикла.
- Для меню используй один цикл `while`, а для показа элементов — цикл `for`.
- Для нумерации используй `enumerate(..., start=1)`.
- Для добавления и удаления используй именно `append()` и `remove()`.
- Перед добавлением и удалением проверяй наличие элемента оператором `in` или
  `not in`.
- Применяй `.strip().lower()` к введённой команде, чтобы ` ADD ` тоже работала.
- Применяй `.strip()` к названию покупки, но сохраняй его исходный регистр.
- Не используй функции, классы, файлы и сторонние библиотеки.
- Имена переменных должны соответствовать стилю `snake_case`.

## Критерии готовности

- Файл называется `main.py` и запускается без ошибок.
- Все четыре команды работают и меню повторяется после каждой операции.
- Дубликат не добавляется в список второй раз.
- Удаление существующего элемента работает.
- Попытка удалить отсутствующий элемент не вызывает ошибку.
- Пустой список отображается отдельным сообщением.
- Непустой список выводится в порядке добавления с нумерацией от единицы.
- После списка правильно выводится количество элементов.
- Команда `exit` корректно завершает программу.
- Неизвестная команда обрабатывается без завершения программы.
- Ты можешь объяснить, чем список отличается от одной обычной переменной.

## Сценарий самопроверки

Выполни последовательно:

1. `show` для пустого списка.
2. Добавь `Хлеб` и `Молоко`.
3. Попробуй снова добавить `Хлеб`.
4. Выполни `show` и проверь номера и количество.
5. Удали `Хлеб`.
6. Попробуй удалить `Сыр`, которого нет.
7. Введи неизвестную команду `hello`.
8. Выполни `show`, затем `exit`.
