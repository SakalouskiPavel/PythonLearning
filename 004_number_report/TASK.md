# Задача 004 — «Числовой отчёт»

## Цель

Познакомиться с циклом `for`, функцией `range()` и переменными-накопителями.
Программа несколько раз выполнит один блок кода и соберёт статистику по числам.

## Условие

Создай файл `main.py`. Программа должна запросить положительное целое число `N`,
а затем:

1. Перебрать все целые числа от `1` до `N` включительно.
2. Для каждого числа вывести само число и его квадрат.
3. Во время того же цикла вычислить сумму всех чисел от `1` до `N`.
4. Посчитать количество чётных чисел в этом диапазоне.
5. После цикла вывести найденную сумму и количество чётных чисел.

Пример для `N = 5`:

```text
Введи положительное целое число: 5
1 -> 1
2 -> 4
3 -> 9
4 -> 16
5 -> 25
Сумма чисел: 15
Количество чётных чисел: 2
```

Формулировки сообщений можно немного изменить, но все значения должны быть
понятны из вывода.

## Темы и ключевые слова для изучения

- цикл `for`;
- функция `range(start, stop)`;
- правая граница `range()` не включается;
- переменная цикла;
- тело цикла и отступы;
- переменная-накопитель;
- сокращённое присваивание `+=`;
- возведение в степень `**`;
- повторное использование условия `number % 2 == 0`;
- f-строки внутри цикла.

## Ограничения

- Используй один цикл `for` и один вызов `range()`.
- Цикл должен выполнить всю работу по выводу квадратов, вычислению суммы и
  подсчёту чётных чисел.
- Не используй готовую функцию `sum()` и математическую формулу суммы ряда.
- Не записывай числа диапазона вручную.
- Для квадрата используй оператор `**`.
- Для вывода числа и его квадрата используй f-строку.
- Пока можно считать, что пользователь вводит целое число `N >= 1`.
- Не используй списки, функции и сторонние библиотеки.

## Подсказка о накопителях

До начала цикла подготовь две переменные со значением `0`: одну для суммы, другую
для количества чётных чисел. На каждой итерации обновляй их при необходимости.

Это общая идея, а имена переменных и точную структуру кода выбери самостоятельно.

## Критерии готовности

- Файл называется `main.py` и запускается без ошибок.
- Обрабатываются все числа от `1` до `N` включительно без пропусков.
- Для каждого числа правильно выводится квадрат.
- Сумма и количество чётных чисел верны для разных значений `N`.
- В решении используется один цикл `for`.
- Накопители создаются до цикла, изменяются внутри него и выводятся после цикла.
- Код имеет правильные отступы и понятные имена переменных в стиле `snake_case`.
- Ты можешь объяснить, почему для включения `N` требуется верхняя граница `N + 1`.

## Самопроверка

Проверь программу как минимум на этих значениях:

| `N` | Сумма | Количество чётных | Последний квадрат |
|---:|---:|---:|---:|
| `1` | `1` | `0` | `1` |
| `2` | `3` | `1` | `4` |
| `5` | `15` | `2` | `25` |
| `10` | `55` | `5` | `100` |

## Дополнительное задание (необязательно)

Внутри того же цикла вычисли ещё и сумму квадратов всех чисел от `1` до `N`.
После цикла выведи результат. Например, для `N = 3` сумма квадратов равна
`1 + 4 + 9 = 14`.

Ограничение на один цикл продолжает действовать.
