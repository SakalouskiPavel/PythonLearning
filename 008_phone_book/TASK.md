# Задача 008 — «Телефонная книга»

## Цель

Научиться хранить связанные пары данных в словаре и применять функции для
добавления, поиска и удаления записей. Эта задача отделяет логику работы с
данными от консольного интерфейса.

## Условие

Создай файл `main.py` с телефонной книгой. Она хранит имя контакта и его телефон:

```python
contacts = {
    "Анна": "+375291112233",
    "Макс": "+375292223344",
}
```

При запуске словарь должен быть пустым. Программа многократно предлагает команды:

```text
add     — добавить контакт
find    — найти контакт по имени
show    — показать все контакты
remove  — удалить контакт
exit    — завершить программу
```

## Обязательные функции

Определи до `main()` три функции. Они не должны использовать `input()` и
`print()`.

### `add_contact(contacts, name, phone)`

- Если имя уже есть в словаре, ничего не изменяет и возвращает `False`.
- Иначе добавляет пару `name: phone` и возвращает `True`.

### `find_contact(contacts, name)`

- Возвращает телефон найденного контакта.
- Если имени нет, возвращает `None`.
- Для поиска используй метод `get()`.

### `remove_contact(contacts, name)`

- Если имя существует, удаляет запись оператором `del` и возвращает `True`.
- Если имени нет, ничего не изменяет и возвращает `False`.

## Поведение команд

### `add`

Запросить имя и телефон, удалить пробелы по краям через `strip()`, вызвать
`add_contact()` и сообщить, был ли контакт добавлен. Дубликат имени не должен
заменять существующий телефон.

### `find`

Запросить имя, вызвать `find_contact()` и вывести найденный телефон либо сообщение
`Контакт не найден.`.

### `show`

Если словарь пуст, вывести `Телефонная книга пуста.`. Иначе вывести контакты по
одному на строке в алфавитном порядке имён:

```text
Анна: +375291112233
Макс: +375292223344
```

Для перебора используй `contacts.items()`, а для сортировки — `sorted()`.

### `remove`

Запросить имя, вызвать `remove_contact()` и сообщить об успешном удалении либо об
отсутствии контакта.

### `exit`

Вывести `До встречи!` и завершить цикл через `break`.

Неизвестная команда должна выводить понятное сообщение и возвращать пользователя
к меню.

## Темы и ключевые слова для изучения

- словарь: `dict` и литерал `{}`;
- пары «ключ — значение»;
- добавление и изменение через `dictionary[key] = value`;
- проверка ключа оператором `in`;
- методы `get()` и `items()`;
- удаление оператором `del`;
- перебор пар: `for key, value in dictionary.items()`;
- сортировка функцией `sorted()`;
- логические результаты функций: `True` и `False`;
- передача изменяемого объекта в функцию;
- разделение логики и пользовательского интерфейса;
- повторение `main()` и `if __name__ == "__main__"`.

## Ограничения

- Создай пустой словарь внутри `main()` до начала цикла меню.
- Используй все три обязательные функции с указанными именами.
- Эти функции работают только с переданными данными и ничего не читают и не
  печатают.
- Команды `add`, `find` и `remove` обязаны вызывать соответствующие функции.
- Для поиска используй `get()`, для удаления — `del`.
- При `show` используй `items()` и `sorted()`.
- Команду обрабатывай после `.strip().lower()`; имена и телефоны — после
  `.strip()`, сохраняя их регистр.
- Не используй глобальный словарь, классы, файлы и сторонние библиотеки.
- Используй `snake_case`, две пустые строки между функциями и точку входа
  `if __name__ == "__main__":`.

## Критерии готовности

- Файл называется `main.py` и запускается без ошибок.
- Контакт добавляется и затем находится по имени.
- Дубликат имени не перезаписывает исходный телефон.
- Существующий контакт удаляется, отсутствующий обрабатывается без ошибки.
- Пустая книга отображается отдельным сообщением.
- Контакты выводятся в алфавитном порядке.
- Неизвестная команда не завершает программу.
- `exit` корректно завершает программу.
- Функции можно импортировать и проверить без запуска меню.
- Ты можешь объяснить, почему словарь удобнее списка для поиска телефона по имени.

## Сценарий самопроверки

1. Выполни `show` для пустой книги.
2. Добавь `Макс` с телефоном `222`.
3. Добавь `Анна` с телефоном `111`.
4. Попробуй снова добавить `Макс` с телефоном `999`.
5. Найди `Макс`: должен остаться телефон `222`.
6. Найди отсутствующий контакт `Иван`.
7. Выполни `show`: `Анна` должна выводиться раньше `Макс`.
8. Удали `Макс`, затем попробуй удалить его ещё раз.
9. Введи неизвестную команду `hello`, затем `exit`.
