# Задача 001 — «Привет, Python!»

## Цель

Создать и самостоятельно запустить первую программу на Python. Эта задача
познакомит тебя с Python-файлом, выполнением кода и выводом текста в терминал.

## Условие

Создай в этой папке файл `main.py`. При запуске программа должна вывести ровно
две строки:

```text
Hello, World!
Я изучаю Python!
```

Затем добавь в начало файла однострочный комментарий, кратко объясняющий, что
делает программа. Комментарий не должен появляться в выводе.

Запусти программу из терминала командой:

```powershell
python main.py
```

Если команда `python` недоступна, попробуй:

```powershell
py main.py
```

## Темы и ключевые слова для изучения

- установка Python и проверка версии: `python --version` или `py --version`;
- исходный файл и расширение `.py`;
- интерпретатор Python;
- функция `print()`;
- строковый литерал (`str`), кавычки;
- последовательное выполнение инструкций;
- однострочный комментарий `#`;
- запуск программы из терминала;
- кодировка UTF-8.

## Ограничения

- Не используй сторонние библиотеки.
- Не копируй вывод вручную: обе строки должна напечатать программа.
- Для каждой строки используй отдельный вызов `print()`.

## Критерии готовности

- В папке есть файл `main.py`.
- Файл запускается без ошибок.
- Вывод в точности совпадает с указанным в условии, включая регистр, пробелы и
  знаки препинания.
- В начале файла есть понятный однострочный комментарий.
- Ты можешь своими словами объяснить назначение `print()`, строк и комментария.

## Дополнительное задание (необязательно)

Добавь третью строку с текущей версией Python, но не вводи номер версии вручную.
Найди в стандартной библиотеке способ получить его во время выполнения программы.

Если выполняешь дополнительное задание, исходные две строки и их порядок должны
остаться без изменений.
