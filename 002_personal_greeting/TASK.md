# Задача 002 — «Персональное приветствие»

## Цель

Создать интерактивную программу, которая получает данные от пользователя,
сохраняет их в переменных и использует в вычислениях и тексте.

## Условие

Создай в этой папке файл `main.py`. Программа должна:

1. Спросить имя пользователя.
2. Спросить его возраст в полных годах.
3. Поздороваться с пользователем по имени.
4. Вычислить и сообщить, сколько лет ему исполнится в следующем году.

Пример диалога:

```text
Как тебя зовут? Анна
Сколько тебе лет? 25
Привет, Анна!
В следующем году тебе будет 26.
```

Текст вопросов можно немного изменить, но итоговые сообщения должны содержать
введённое имя и правильно вычисленный возраст.

## Темы и ключевые слова для изучения

- функция `input()`;
- переменные и присваивание `=`;
- понятные имена переменных;
- типы данных `str` и `int`;
- преобразование строки в число: `int()`;
- арифметические операторы, сложение `+`;
- f-строки: `f"...{variable}..."`;
- разница между текстом `"25"` и числом `25`;
- порядок выполнения инструкций.

## Ограничения

- Используй ровно два вызова `input()`: один для имени и один для возраста.
- Возраст следующего года должен вычисляться программой, а не быть записан в
  коде заранее.
- Преобразуй введённый возраст в `int`.
- Для двух итоговых сообщений используй f-строки.
- Не используй сторонние библиотеки.
- Пока не нужно обрабатывать ошибочный ввод: можно считать, что пользователь
  всегда вводит целое неотрицательное число.

## Критерии готовности

- В папке есть файл с точным именем `main.py`.
- Программа запускается без ошибок при корректном вводе.
- Имя из ввода появляется в приветствии.
- Возраст в следующем году вычисляется правильно для разных исходных значений,
  включая `0`.
- В коде есть переменные для имени, текущего возраста и возраста в следующем
  году.
- Ты можешь объяснить, почему результат `input()` требуется преобразовать через
  `int()` перед сложением с единицей.

## Самопроверка

Запусти программу несколько раз и проверь как минимум эти случаи:

| Имя | Возраст | Ожидаемый возраст в следующем году |
|---|---:|---:|
| Анна | 25 | 26 |
| Макс | 0 | 1 |
| Python-разработчик | 99 | 100 |

## Дополнительное задание (необязательно)

После основных сообщений выведи, сколько полных десятилетий уже прожил
пользователь. Например, для возраста `25` результат равен `2`.

Для этого самостоятельно изучи оператор целочисленного деления `//`. Основная
часть программы должна продолжать работать без изменений.
