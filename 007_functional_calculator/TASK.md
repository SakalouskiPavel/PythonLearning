# Задача 007 — «Функциональный калькулятор»

## Цель

Научиться разбивать программу на функции, передавать им аргументы и возвращать
результат с помощью `return`. Заодно познакомиться с числами типа `float` и
точкой входа программы.

## Условие

Создай файл `main.py` с консольным калькулятором. Программа должна поддерживать
команды:

```text
+     — сложение
-     — вычитание
*     — умножение
/     — деление
exit  — завершение программы
```

После выбора арифметической операции программа запрашивает два числа, выполняет
операцию и выводит результат. Затем снова запрашивает команду.

Пример диалога:

```text
Операция (+, -, *, / или exit): +
Первое число: 2.5
Второе число: 4
Результат: 6.5
Операция (+, -, *, / или exit): exit
До встречи!
```

## Обязательные функции

Определи до функции `main()` четыре отдельные функции:

```python
add(first_number, second_number)
subtract(first_number, second_number)
multiply(first_number, second_number)
divide(first_number, second_number)
```

Каждая функция принимает два числа и возвращает результат через `return`.
Арифметические функции не должны использовать `input()` или `print()`.

Функция `divide()` должна проверять делитель. Если второе число равно нулю, она
возвращает `None` вместо выполнения деления. Основная программа при получении
`None` выводит понятное сообщение об ошибке и продолжает работу.

## Функция `main()`

Весь пользовательский интерфейс — цикл меню, `input()` и `print()` — размести в
отдельной функции `main()`. В конце файла добавь точку входа:

```python
if __name__ == "__main__":
    main()
```

Неизвестная операция должна приводить к сообщению `Неизвестная операция.`. В
этом случае числа запрашивать не нужно: программа сразу возвращается к меню.

Числа считывай через `float()`, чтобы калькулятор работал как с целыми, так и с
дробными значениями. Пока можно считать, что пользователь вводит корректные
числа.

## Темы и ключевые слова для изучения

- объявление функции: `def`;
- параметры и аргументы функции;
- вызов функции;
- оператор `return`;
- разница между `return` и `print()`;
- значение `None` и проверка `is None`;
- локальные переменные и область видимости;
- тип `float`;
- функция `main()`;
- специальная переменная `__name__`;
- конструкция `if __name__ == "__main__"`;
- принцип одной ответственности: вычисления отдельно от интерфейса.

## Ограничения

- Используй все четыре функции с указанными именами и параметрами.
- Каждая операция выполняется соответствующей функцией, а не прямо внутри меню.
- Арифметические функции только вычисляют и возвращают значение; они ничего не
  читают и не печатают.
- Интерфейс находится внутри `main()`.
- Для повторения меню используй один цикл `while`.
- Перед запросом чисел проверь, что команда является допустимой.
- Деление на ноль не должно аварийно завершать программу.
- Используй точку входа `if __name__ == "__main__":`.
- Соблюдай стиль `snake_case` и отделяй определения функций двумя пустыми строками.
- Не используй сторонние библиотеки.

## Критерии готовности

- Файл называется `main.py` и запускается без ошибок.
- Все четыре операции правильно работают для целых и дробных чисел.
- Порядок операндов при вычитании и делении не перепутан.
- Деление на ноль выводит ошибку, после которой калькулятор продолжает работу.
- Неизвестная команда не приводит к запросу двух чисел.
- `exit` завершает программу с прощальным сообщением.
- Вычислительные функции возвращают результаты и не взаимодействуют с консолью.
- Вызов `main()` защищён проверкой `__name__`.
- Ты можешь объяснить, зачем вычисления вынесены из `main()` и чем результат
  функции отличается от напечатанного текста.

## Сценарий самопроверки

Проверь за один запуск:

| Команда | Первое число | Второе число | Ожидаемый результат |
|---|---:|---:|---:|
| `+` | `2` | `3` | `5.0` |
| `-` | `10` | `4` | `6.0` |
| `*` | `2.5` | `4` | `10.0` |
| `/` | `7` | `2` | `3.5` |
| `/` | `5` | `0` | сообщение об ошибке |

Затем введи неизвестную операцию `hello` и заверши программу командой `exit`.
